🚀 DÉMARRAGE EN 3 ÉTAPES
═══════════════════════════════════════════════════════════════════════════════

ÉTAPE 1: Installation (une seule fois)
───────────────────────────────────────
  pip install -r backend/requirements.txt
  cd frontend && npm install

ÉTAPE 2: Lancement (3 terminaux)
─────────────────────────────────
  Terminal 1: python backend/orchestrator.py
  Terminal 2: python backend/api_server.py
  Terminal 3: cd frontend && npm run dev

ÉTAPE 3: Utilisation
────────────────────
  1. Ouvrir http://localhost:3101
  2. Cliquer "Connect Backend"
  3. Les métriques se mettent à jour en temps réel

🐛 DÉPANNAGE
═══════════════════════════════════════════════════════════════════════════════

PROBLÈME: "Backend offline"
───────────────────────────
  → Vérifier: python backend/api_server.py tourne
  → Tester: curl http://localhost:5000/api/health

PROBLÈME: "Port 5000 already in use"
────────────────────────────────────
  → Windows: netstat -ano | findstr :5000
  → Tuer: taskkill /PID <PID> /F

PROBLÈME: "CORS error"
──────────────────────
  → Installer: pip install flask-cors
  → Vérifier: flask-cors dans requirements.txt

PROBLÈME: "Module not found"
────────────────────────────
  → Installer: pip install -r backend/requirements.txt

PROBLÈME: Données périmées
──────────────────────────
  → Vérifier: python backend/orchestrator.py tourne
  → Vérifier: Fichiers JSON dans output/
